## Merchant - Delete

Delete merchant.

> Request (POS -> weorder)

```
HTTP/1.1 DELETE https://api.weorder.com/pos/v1/merchant-groups/1/merchants/101
```

### HTTP Request

`HTTP/1.1 DELETE https://api.weorder.com/pos/v1/merchant-groups/{merchantGroupNo}/merchants/{merchantNo}`

> Response: no content

### HTTP Response (success)

`HTTP/1.1 204`
